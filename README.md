## How to run
Apply for a 7-day free trial (coming soon) token and put your token in 'token' in index.ts.

0、先看启动的问题

1、run: 
```
npm install
npm start
```
2、用企微微信扫描二维码

3、在code.txt中输入设备验证码

4、正常成功登录
